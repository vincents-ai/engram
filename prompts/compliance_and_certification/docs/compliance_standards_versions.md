# Compliance Standards and Versions

This document lists the versions of compliance standards and regulations referenced in the auditing prompts within this project. Versions have been validated against the latest available from official sources as of August 2025.

## Cybersecurity and Information Security

- **ISO/IEC 27001**: 2022 (Information Security Management Systems)
- **ISO/IEC 27002**: 2022 (Security Controls)
- **CIS Critical Security Controls**: v8.1
- **NIST Cybersecurity Framework (CSF)**: 2.0
- **NIST SP 800-61**: 2010 (Computer Security Incident Handling)
- **NIST SP 800-34**: 2010 (Contingency Planning)
- **NIST SP 800-207**: 2020 (Zero Trust Architecture)
- **NIST SP 800-53**: 2013 (Security and Privacy Controls)
- **NIST SP 800-60**: 2008 (Guide for Mapping Types of Information)
- **NIST Risk Management Framework (RMF)**: Current (integrated with CSF 2.0)
- **ISO 22301**: 2019 (Business Continuity Management)
- **ISO 31000**: 2018 (Risk Management)

## Data Protection and Privacy

- **GDPR (General Data Protection Regulation)**: EU Regulation 2016/679 (as amended)
- **CCPA/CPRA (California Consumer Privacy Act)**: Current (with 2020 amendments)
- **PIPEDA (Personal Information Protection and Electronic Documents Act)**: Current (Canadian)
- **LGPD (Lei Geral de Proteção de Dados)**: Current (Brazilian data protection law)
- **POPIA (Protection of Personal Information Act)**: Current (South African data protection)
- **DPA 2018 (UK Data Protection Act)**: Current (UK implementation of GDPR)

## Notes

- Versions marked as "Current" indicate standards that do not use version numbering or are continuously updated without formal version releases.
- All versions were validated against the latest available from official sources as of August 2025.
- For standards with amendments or corrigenda, the base version is listed with notes on updates where applicable.
- Auditor certifications require ongoing CPE/CPD credits and periodic renewal.
- This document should be reviewed annually to ensure compliance prompts reference the latest standards and certifications.
