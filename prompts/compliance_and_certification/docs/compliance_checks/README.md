## Report Navigation

Each report includes:
- **Previous/Next Report Links**: Navigate chronologically between audit reports

## GitHub Integration

The Auditron agent automatically creates GitHub issues for:
- **Non-Compliant** findings (high priority, immediate action required)
- **Observations** (lower priority, recommendations for improvement)

Issues are tagged with the relevant compliance framework and assigned to appropriate project teams.
