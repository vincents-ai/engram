This collection provides comprehensive audit checkpoint prompts for ITIL v4 Service Management and CSA STAR Cloud Security assessments.

## Context and Background

### ITIL v4 Service Management
**Effective Date**: 2019 (current version)  
**Scope**: IT service management across all organization types  
**Key Principles**: Focus on value, Collaborate and promote visibility, Progress iteratively with feedback, Think and work holistically, Keep it simple and practical  
**Structure**: Service Value System, Four Dimensions, 34 Practices, 3 Guiding Principles  
**Benefits**: Improved service quality, better customer experience, enhanced operational efficiency, reduced costs

### CSA STAR Cloud Security
**Effective Date**: 2019 (current version)  
**Scope**: Cloud service providers and cloud customers  
**Key Domains**: Cloud Architecture, Governance & Risk Management, Legal, Information Management, Portability & Interoperability, Traditional Security, Data Center Operations, Incident Response, Application Security, Encryption, Identity & Access Management, Virtualization, Network Security  
**Assessment Levels**: Level 1 (Self-assessment), Level 2 (Third-party attestation), Level 3 (Continuous monitoring)  
**Benefits**: Standardized cloud security assessment, improved transparency, better risk management, enhanced trust

## Regulatory Context
- **ITIL v4**: Globally recognized best practice framework for IT service management
- **CSA STAR**: Industry-led cloud security assurance program
- **Integration**: ITIL practices can be applied to cloud service management
- **Certification**: STAR Level 1, 2, and 3 certifications available
