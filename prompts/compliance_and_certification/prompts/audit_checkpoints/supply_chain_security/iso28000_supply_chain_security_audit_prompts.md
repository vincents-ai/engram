# ISO 28000 Supply Chain Security Management System Audit Prompts

This collection provides comprehensive audit checkpoint prompts for ISO 28000:2022 Supply Chain Security Management implementation and assessment.

## Context and Background

### ISO 28000:2022 Security and Resilience - Security Management Systems for the Supply Chain
**Effective Date**: December 15, 2022 (current version)  
**Scope**: Security management for organizations involved in the supply chain  
**Key Principles**: Risk-based approach, stakeholder involvement, continual improvement, leadership commitment  
**Structure**: Requirements covering security planning, implementation, monitoring, and improvement  
**Benefits**: Enhanced supply chain security, reduced security incidents, improved stakeholder confidence, regulatory compliance

## Regulatory Context
- **ISO 28000**: International standard for supply chain security
- **Voluntary Standard**: Provides framework for security management
- **Global Application**: Applicable to all supply chain participants
- **Risk-Based**: Emphasizes security risk assessment and mitigation

## ISO 28000 Supply Chain Security Assessment

**Security Threat Categories:**
- **Physical Threats**: Theft, tampering, sabotage, unauthorized access
- **Cyber Threats**: Hacking, data breaches, system disruptions
- **Human Threats**: Insider threats, social engineering, negligence
- **Logistical Threats**: Hijacking, diversion, counterfeiting
- **Environmental Threats**: Natural disasters, infrastructure failures

**Security Control Framework:**
- **Preventive Controls**: Access controls, surveillance, security barriers
- **Detective Controls**: Monitoring systems, audit trails, inspections
- **Responsive Controls**: Incident response, recovery procedures
- **Recovery Controls**: Business continuity, backup systems
