# ISO 14644 Cleanrooms Audit Prompts

This collection provides comprehensive audit checkpoint prompts for ISO 14644 series Cleanrooms and Associated Controlled Environments implementation and assessment.

## Context and Background

### ISO 14644 Series: Cleanrooms and Associated Controlled Environments
**Effective Date**: Various dates for different parts (1999-2020)  
**Scope**: Classification, testing, and monitoring of cleanrooms and controlled environments  
**Key Principles**: Airborne particulate cleanliness, classification system, testing methods, monitoring requirements  
**Structure**: Multiple parts covering classification, testing, design, operation, and metrology  
**Benefits**: Controlled contamination levels, product quality assurance, regulatory compliance, operational efficiency

## Regulatory Context
- **ISO 14644**: International standard for cleanroom classification and control
- **Industry Applications**: Pharmaceuticals, electronics, aerospace, healthcare
- **Regulatory Compliance**: Required for GMP, GLP, and other controlled environments
- **Global Recognition**: Accepted worldwide for cleanroom validation

**ISO 14644 Compliance Assessment:**

1. **ISO 14644-1: Classification of Air Cleanliness**
   - Evaluate cleanroom classification methodology
   - Assess airborne particulate concentration limits
   - Review classification measurement procedures
   - Validate classification documentation

2. **ISO 14644-2: Monitoring to Provide Evidence of Cleanroom Performance**
   - Assess monitoring plan development
   - Review monitoring location selection
   - Validate monitoring frequency determination
   - Confirm monitoring data analysis procedures

3. **ISO 14644-3: Test Methods**
   - Evaluate particle counting procedures
   - Assess airflow velocity measurement
   - Review air pressure differential testing
   - Validate filter integrity testing

4. **ISO 14644-4: Design, Construction and Start-Up**
   - Assess cleanroom design requirements
   - Review construction material selection
   - Validate installation qualification procedures
   - Confirm operational qualification processes

5. **ISO 14644-5: Operations**
   - Evaluate operating procedures development
   - Assess personnel training and gowning
   - Review contamination control measures
   - Validate maintenance and cleaning procedures

6. **ISO 14644-7: Separative Devices (Clean Air Hoods, Glove Boxes, Isolators)**
   - Assess separative device classification
   - Review device performance testing
   - Validate device monitoring procedures
   - Confirm device maintenance requirements

7. **ISO 14644-8: Classification of Airborne Molecular Contamination**
   - Evaluate molecular contamination assessment
   - Assess AMC measurement procedures
   - Review AMC control strategies
   - Validate AMC monitoring requirements

## ISO 14644 Cleanroom Assessment

**Cleanroom Classes:**
- **ISO Class 1**: ≤10 particles/m³ (≥0.1 µm)
- **ISO Class 2**: ≤100 particles/m³ (≥0.1 µm)
- **ISO Class 3**: ≤1,000 particles/m³ (≥0.1 µm)
- **ISO Class 4**: ≤10,000 particles/m³ (≥0.1 µm)
- **ISO Class 5**: ≤100,000 particles/m³ (≥0.1 µm)
- **ISO Class 6**: ≤1,000,000 particles/m³ (≥0.1 µm)
- **ISO Class 7**: ≤10,000,000 particles/m³ (≥0.1 µm)
- **ISO Class 8**: ≤100,000,000 particles/m³ (≥0.1 µm)
- **ISO Class 9**: ≤1,000,000,000 particles/m³ (≥0.1 µm)

**Contamination Control Measures:**
- **HEPA/ULPA Filters**: High-efficiency particulate air filters
- **Airflow Patterns**: Laminar, turbulent, or mixed flow
- **Personnel Practices**: Gowning, training, behavior controls
- **Material Controls**: Cleaning, maintenance, contamination prevention
