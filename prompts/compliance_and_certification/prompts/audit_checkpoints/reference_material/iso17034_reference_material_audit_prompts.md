# ISO 17034 Reference Material Producer Audit Prompts

This collection provides comprehensive audit checkpoint prompts for ISO 17034:2016 Conformity Assessment implementation and assessment.

## Context and Background

### ISO 17034:2016 General Requirements for the Competence of Reference Material Producers
**Effective Date**: November 15, 2016 (current version)  
**Scope**: Organizations producing reference materials for calibration and testing  
**Key Principles**: Impartiality, competence, consistency, traceability, metrological traceability  
**Structure**: Requirements covering general requirements, structural requirements, resource requirements, process requirements, management system requirements  
**Benefits**: Reliable reference materials, enhanced measurement quality, improved testing accuracy, regulatory compliance

## Regulatory Context
- **ISO 17034**: International standard for reference material producers
- **Accreditation Requirement**: Reference material producers must be accredited to ISO 17034
- **Measurement Quality**: Ensures traceability and reliability of measurements
- **Global Recognition**: Accepted worldwide for reference material competence

## ISO 17034 Reference Material Assessment

**Reference Material Categories:**
- **Certified Reference Materials (CRMs)**: With certified values and uncertainties
- **Reference Materials (RMs)**: With property values and associated uncertainties
- **Quality Control Materials**: For laboratory quality control
- **Calibration Materials**: For instrument calibration

**Quality Assurance Processes:**
- **Characterization**: Determination of property values
- **Homogeneity Testing**: Ensuring material uniformity
- **Stability Testing**: Monitoring long-term stability
- **Value Assignment**: Establishing certified/reference values
